API Documentation

Base URL

https://raw.githubusercontent.com/uthumany/Ambient-sounds-API/master/

Response Format

All data is returned in standard JSON format, including file names, direct URLs, and categories.

{
  "name": "Rain Sounds",
  "url": "https://.../rain.mp3",
  "category": "Nature"
}

Available Endpoints

Method Endpoint Description
GET /sounds.json Fetch all available ambient sounds and recitations.
GET /{filename}.mp3 Direct stream/download link for a specific audio file.
