# Ambient Sounds API 🌬️🎵

A high-quality collection of ambient sounds and Islamic recitations provided via a responsive JSON API. This repository hosts audio files and provides a structured way to access them for developers building relaxation, meditation, or background atmosphere applications.

## 🚀 Features
- **High-Quality Audio**: Curated collection of nature sounds and spiritual recitations.
- **JSON API**: Easy-to-parse endpoint for all assets.
- **SEO Optimized**: Documentation designed for search engine visibility.
- **Responsive UI**: Interactive documentation page with live audio previews.

## 🛠️ API Usage

### Base URL
`https://raw.githubusercontent.com/uthumany/Ambient-sounds-API/master/`

### Endpoints
- **Get All Sounds**: `GET /sounds.json`
- **Stream Audio**: `GET /{filename}.mp3`

## 📂 Content Categories
- **Nature/Ambient**: Forest, Wind, Ocean, Rain, Snow Storms.
- **Islamic**: Quranic recitations and Nasheeds.
